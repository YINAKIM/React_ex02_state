1.컴포넌트란 무엇인가?
함수로 작성되어 반환값에 렌더링할 요소를 반환하는 것. 컴포넌트의 이름은 대문자로 시작한다.
ReactDOM.render(); 에 첫번째 인자로 전달되어 렌더링 할 것들을 반환하는 것.


2.JSX란?
react에서 컴포넌트를 만들 때 쓰는 문법
Javascript + HTML 이 조합된 형태이면서 독자적인 문법을 가진다.
컴포넌트함수에서 return ~~~ ; 까지가 JSX다. 이 문법을 사용하여 컴포넌트를 작성한다.

3. props란?
컴포넌트에게 데이터를 전달할 때 사용하는 것.
함수에 전달되는 인자의 개념으로 이해하면되고,
여러개의 속성을 가지는 props가 전달된다면 props.property1 이런식으로 꺼내서 사용한다.
렌더링되는 Object의 속성을 출력하는 것으로, props를 사용하여 컴포넌트를 효율적으로 재사용할 수 있다.
